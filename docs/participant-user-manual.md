# 被试者产品使用手册

## 1. 产品用途

本系统是一个“研究训练助手”，用于辅助完成文献阅读、论文资料整理、证据检索和研究任务讨论。你可以通过对话提出问题，也可以上传 PDF 文献或导入 arXiv 论文，让系统帮助你提取重点、查找证据和生成分析结果。

## 2. 使用前准备

实验开始前，研究人员会打开系统页面。你只需要使用浏览器中的系统界面，不需要安装软件。

默认访问地址通常为：

```text
http://127.0.0.1:5173
```

如果页面显示“无法连接后端服务”，请不要自行处理，直接告知研究人员。

## 3. 页面组成

系统左侧是功能导航区，主要包含：

- 对话：与研究助手交流，提出研究问题或操作请求。
- 文献：上传 PDF、导入 arXiv 论文、查看解析状态和证据片段。
- 成果：查看系统生成的分析结果、摘要或比较报告。
- 设置：查看系统状态。被试者一般不需要修改设置。

页面顶部会显示当前研究项目名称。实验过程中请按照研究人员指定的项目进行操作。

## 4. 对话功能

进入“对话”页面后，你可以在输入框中输入问题或任务。

常见输入示例：

```text
请总结这篇论文的研究目的、方法和主要结论。
```

```text
请帮我找出文献中与认知负荷有关的证据。
```

```text
请比较两篇论文在实验设计上的差异。
```

发送后，请等待系统回复完成。若回复较慢，请不要重复点击发送。

## 5. 文献上传与分析

进入“文献”页面后，可以使用以下功能：

- 上传 PDF：选择本地 PDF 文件，系统会解析文本内容。
- 导入 arXiv：输入 arXiv 编号或链接，系统会尝试导入论文。
- 快速分析：对已上传或导入的论文生成基础分析。
- 证据检索：输入关键词，查找论文中的相关证据片段。

上传后，系统可能需要一段时间处理文件。请等待任务状态变为完成后再继续操作。

## 6. 查看成果

进入“成果”页面，可以查看系统生成的分析结果。常见成果包括：

- 文献摘要
- 证据整理
- 论文比较
- 研究建议

点击具体成果后，可以查看详细内容。如果研究人员要求记录结果，请按照实验要求复制或填写。

## 7. 使用注意事项

- 请按照研究人员给出的任务顺序操作。
- 不要刷新页面，除非研究人员要求。
- 不要关闭浏览器或终端窗口。
- 不要修改“设置”页面中的内容。
- 如果系统长时间无响应，请告知研究人员。
- 如果出现错误页面，请记录当时正在进行的操作，并告知研究人员。

## 8. 常见问题

### 页面提示无法连接后端服务

说明系统服务可能没有启动或连接中断。请告知研究人员。

### 上传 PDF 后一直显示处理中

PDF 文件可能较大，解析需要等待。如果超过研究人员规定时间仍未完成，请告知研究人员。

### 系统回复内容不完整或不符合预期

可以在对话中继续追问，例如：

```text
请用更简洁的方式重新总结。
```

```text
请给出文献中的具体证据。
```

### 不知道该输入什么

请参考研究人员提供的任务说明，不需要自由探索系统功能。

## 9. 实验结束

完成全部任务后，请停止操作，并告知研究人员。不要自行清空数据、关闭服务或修改设置。
